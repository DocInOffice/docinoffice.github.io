FB2 Add-In 1.0.15

2017-03-24 16:28:00 +0300, Леонид

Привет народ.

Доступна версия FB2 Add-In v1.0.15.

FB2 Add-In - редактор для fb2 файлов, а точнее расширение для Microsoft Word. Хочу чтоб пользователи, которые создают FB2 книги имели мощное средство, как Word, для создания качественного контента. Т.е. чтобы в тексте было меньше ошибок и пунктуация не хромала.

Как работать.

После установки FB2 Add-In вам в окнах «Открытие документ» и «Сохранение документа» будет доступен тип файлов «FB2 Book»:

Так как программа тестовая, то в ней отключено возможность открытия по двойному щелчку в Windows Explorer, чтобы случайно не испортить нормальные fb2 файлы книг.

Чтобы создать новую FB2 книгу, нужно в Windows Explorer вызвать контекстное меню папки и выбрать «FB2 Book» из подменю «Создать»:

Скачать программу можно здесь.

На данный момент доступна только версия для Microsoft Word 2010.

Полноценная реализация программы реальна, но она требует очень много времени. И чтобы укорить процесс я решил создать краудфандинговый проект. Средства потрачу, чтобы сосредоточиться полностью на развитии программы.

За новостями следите здесь, в проекте на planeta.ru, в твиттере и в телеграмм.

Установка на Windows 10

2016-08-05 19:10:00 +0300, Леонид

Замечание по установке.

Походу Microsoft начало закручивать гайки для программ свободно распространяемых через интернет. Мол, все идите к ним в магазин. Поэтому, не смотря на то, что файлы инсталляторов подписаны, на Windows 10 вы можете увидеть красное или синее окно предупреждение.
