FB2 Word Viewer 1.0.1.12

2009-03-18 11:30:00 +0300, Леонид

Здравствуйте.

Дорогие любители электронных книг хочу вам предложить программу FB2 Word Viewer, для просмотра файлов формата Fiction Book 2 (*.fb2).

FB2 Word Viewer представляет из себя расширение для программы Microsoft Word. После его установки файл с расширением .fb2 можно открывать в программе Word. Это возможно сделать через диалог “Открытие документа” (“Open”) или из Проводника (Windows Explorer).

Для того чтобы в Microsoft Word открыть файл с расширением .fb2 следует вызвать диалог “Открытие документа” и в фильтрах файлов выбрать элемент “Fiction Book [Read Only] (*.fb2)”:

Таким образом, в списках файлах будут доступны файлы с расширением .fb2. Затем следует выделить необходимые файлы и нажать кнопку открыть.

Открытие через проводник (Windows Explorer) может иметь два сценария:

  1. Если до установки расширение .fb2 не было связанно ни с одним приложением, то при двойном щелчке на файле с расширением .fb2 он сразу же будет открыт в программе Word:

  1. Если же до установки расширение .fb2 было связанно с каким-либо приложением, то файл можно будет открыть через меню “Открыть с помощью”:

Для удобства просмотра можно использовать режим чтения. Он включается при помощи кнопки в статус панели:
