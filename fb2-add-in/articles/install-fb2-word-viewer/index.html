Установка программы FB2 Add-In

Для установки FB2 Add-In, программы расширения для Microsoft Word, вам нужно зайти на страницу загрузки и выбрать нужный вам файл установки в зависимости от версии Microsoft Word.

Версию Microsoft Word вы можете посмотреть в названии иконки, которая запускает программу.

После того, как вы определились с Word-ом загрузите со страницы загрузки нужный файл установки и запустите его.

По завершению установки программа FB2 Add-In будет интегрирована в Microsoft Word и вы сможете открывать файл fb2 в Microsoft Word.
